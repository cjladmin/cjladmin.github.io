使用手册

《文字互动相关指令》

==功能名称== ==触发指令==(无需输入+号，灰色为必输内容) ==是否需要@== ==功能说明==
联系管理 联系管理 或 反馈bug 需要@ 雷雨会给本人(即开发者)发送通知
戳一戳 双击头像触发戳一戳 无需@ 发送一句戳一戳随机语句
点歌 点歌 或 网易点歌 + 要点的歌名 无需@ 发送您所点的歌曲(默认QQ音乐)
今日人品 今日人品 需要@ 发送随机生成的人气值(切勿当真)
lol台词 lol台词 或 lol励志台词 或 lol语录 无需@ 随机发送一句lol英雄台词
伤感文案 伤感文案 或 伤感语录 或 感人文案 无需@ 随机发送一句伤感语录
笑话 笑话 需要@ 随机发送一句不同类型的笑话
手机号归属地 归属地 + 手机号前7位 需要@ 发送该手机号对应的归属地和运营商
天气 地区 + 天气 需要@ 发送指定地区的实时天气
翻译 翻译 + 需要翻译的内容 需要@ 默认将内容翻译为中文
翻译模式 进入翻译模式 需要@ 发送的所有内容都将翻译为指定语言
古诗词 最美诗词 需要@ 随机发送一句古诗词
历史 历史的今天 需要@ 发送过去历史中今天发生的大事
对联 随机对联 或 对联 或 对子 无需@ 随机发送一副对联
查电费 查电费 + 宿舍号 无需@ 发送当前宿舍的剩余电量(仅支持商大)
老黄历 老黄历 或 今日适宜 或 今日禁忌 无需@ 发送当天的适宜和禁忌
药品说明 查药品 + 要查的药名 无需@ 发送您要查的药品使用说明书
成语解释 查成语 + 要查的成语 需要@ 发送您所查成语的对应解释
lol英雄故事 lol + 英雄名 + 背景故事 无需@ 发送对应的英雄背景故事
舔狗日记 舔狗语录 或 舔狗文案 或 舔狗日记 无需@ 发送随机一句舔狗语录
随机一言 随便说一句 或 随机一言 无需@ 随机发送一句话，比如：动漫中的台词
二次元图片 二次元 或 二次元图片 或 抽老婆 无需@ 随机发送一张二次元萝莉图
疫情查询 地区 + 疫情 无需@ 发送要查地区的具体疫情状况
代码云执行 code 代码语言 [-i] [inputText] + 代码体 无需@ 返回代码的执行结果
发图识番 识番 + 图片 无需@ 返回与图片相似的番剧
每日一句 每日一句 无需@ 每日一句英语小短句
微博热搜 微博热搜 无需@ 发送实时微博热搜榜
新闻 新闻 无需@ 发送央视实时最新新闻

《图片互动相关指令》

全无需@ 触发指令 效果展示
~ 灰色头像 或 黑白头像
~ 诺基亚 + 要显示的内容
~ 鲁迅说 + 要说的内容 这个也违规了放不上。。。

~ 亲 或 亲亲 + QQ号、@某人、自己、图片
~ 贴贴 或 蹭蹭 + QQ号、@某人、自己、图片
~ 顶 或 玩 + QQ号、@某人、自己、图片
~ 拍 或 想吃 + QQ号、@某人、自己、图片
~ 撕 或 手撕 + QQ号、@某人、自己、图片
~ 丢 或 扔 + QQ号、@某人、自己、图片
~ 爬 + QQ号、@某人、自己、图片
~ 精神支柱 + QQ号、@某人、自己、图片

《无指令的触发功能》

功能名称 功能说明
入退群通知 当群成员有变化时，自动发送通知消息
群文件变化 当有成员上传文件时，自动发送通知消息
凌晨打卡 新一天的开始定时发送，附带每日一言
早安语录 每天早上6点定时发送，随机早安文案
历史的今天 每天早上8点定时发送，了解历史
微博热搜榜 每天上午10点定时发送，微博实时热搜
央视新闻 每天中午12点定时发送，央视实时新闻

  • 群管理功能(机器人必须是群管理)：
注：群管为特殊指令，所以开头必须加`/`
【管理群聊】
    禁言:
        /禁 @某人 时间(s)[1,2591999]
        /禁 @某人 缺省时间则随机
        /解 @某人
    踢人：
        /踢 @某人
    踢人并拉黑：
        /黑 @某人
    修改昵称:
        /改 @某人 [昵称]
    修改头衔(需身份是群主):
        /头衔 @某人 [头衔]
        /删头衔 @某人
【加群自动审批】
    查看本群审批词条：
        /审批  
    增加审批词条：
        /审批+ [词条]
    删除审批词条：
        /审批- [词条]

俄罗斯轮盘游戏

发送 -> /俄罗斯轮盘帮助获取帮助信息

该文字游戏食用指南：

获取金币：
    发送 -->   /轮盘签到
开启游戏：
    发送 -->   /装弹[子弹数] [金额] [@xxx]
    (@为指定人，不@时则所有人都可接受决斗)
接受对决：
    发送 -->   /接受对决/接受挑战/拒绝决斗/拒绝挑战
开枪对决：
    发送 -->   /开枪/咔/嘭/嘣
    (轮流开枪，30秒未开枪另一方可使用‘结算’命令结束对决并胜利)
结算结果：
    发送 -->   /结算
    (当某一方30秒未开枪，可使用该命令强行结束对决并胜利)
查看战绩：
    发送 -->   /我的战绩
查询金币：
    发送 -->   /我的金币
查看排行榜：
    发送 -->   /金币排行/胜场排行/败场排行/欧洲人排行/慈善家排行
【注：同一时间群内只能有一场对决】
